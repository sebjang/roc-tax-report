Section 01 — Mathematical Proof

ROC가 이익·잉여금이 아님을 증명하는 수학

미국 세법에서 "배당(dividend)"은 오직 이익잉여금(Earnings and Profits, E&P) 범위 내에서만 발생합니다. ROC는 E&P가 0 또는 음수인 상황에서 지급되는 분배금의 잔여분으로, 수학적으로 이익 및 잉여금과 완전히 분리됩니다.

핵심 정의 공리 (IRC §316 기반)

AXIOM — IRC §316(a)
Dividend := Distribution × 𝟙[E&P_current > 0 ∨ E&P_accumulated > 0] ROC := Distribution − Dividend = Distribution × 𝟙[E&P = 0] (잉여금 소진 후 잔여분)
𝟙[·] = 지시함수(indicator function). E&P가 존재하면 1, 없으면 0.
ROC는 오직 E&P가 완전히 소진되었을 때에만 정의됨 → 이익의 함수가 아님.

분배금 3단계 분류 정리

THEOREM — 분배금 성격 결정 공식
Distribution D에 대하여 (IRC §301(c)): Step 1. Dividend = min(D, E&P_current + E&P_accumulated) → 과세 배당소득 [IRC §301(c)(1)] Step 2. ROC = min(D − Step1, Adjusted_Basis) → 비과세 자본 반환 [IRC §301(c)(2)] → Adjusted_Basis ↓ (감소) Step 3. Gain = max(D − Step1 − Step2, 0) → 자본이득 [IRC §301(c)(3)]
D
= 분배금 총액 (Total Distribution)
E&P_current
= 당해 연도 이익잉여금 (IRC §312)
E&P_accumulated
= 누적 이익잉여금 (1913년 또는 법인 설립 이후)
Adjusted_Basis
= 주주의 주식 취득 원가 (IRC §1001)
■ PROOF — ROC ≢ Earnings (ROC는 이익이 아님)
가정: D = 100, E&P_current = 0, E&P_accumulated = 0, Basis = 80 Step 1: Dividend = min(100, 0+0) = 0 ← 이익/잉여금 = 0이므로 배당 없음 Step 2: ROC = min(100-0, 80) = 80 ← Basis 소진 Step 3: Gain = max(100-0-80, 0) = 20 ← 자본이득 따라서: ROC = 80 ∧ Dividend = 0 ROC > 0 이지만 Earnings = 0 이므로, ROC ≢ f(Earnings) → ROC는 이익의 함수가 아님 □
∎ Q.E.D.

항등식: E&P와 ROC의 배타적 관계

IDENTITY — E&P ∩ ROC = ∅
Dividend_t + ROC_t + Gain_t ≡ Distribution_t (∀t) where: Dividend_t = f(E&P) → 이익의 함수 ROC_t = f(Basis) → 취득원가의 함수 (E&P 無관) Gain_t = f(Basis − ROC) → 잔여 원가 초과분 ∴ E&P와 ROC는 상호 배타적(mutually exclusive): E&P > 0 → 전액 Dividend 처리 → ROC = 0 E&P = 0 → Dividend = 0 → ROC = min(D, Basis)

Section 02 — E&P Calculation

E&P(이익잉여금) 계산 공식 — 과세소득 ≠ E&P

E&P는 GAAP 회계 이익도 아니고 세무 과세소득도 아닙니다. E&P는 세법 고유의 개념으로, 과세소득에서 출발하여 IRC §312에 따른 수십 가지 조정을 거쳐 산출됩니다. 이 괴리가 ROC 발생의 핵심 원인입니다.

E&P COMPUTATION — IRC §312
E&P_current = Taxable_Income [ + 가산 항목 ] + 비과세 소득 (면세 채권 이자 등) [IRC §103] + 과세소득에서 공제된 연방소득세 [IRC §312(f)] + 비용불인정 지출 (식대 50% 등) [IRC §274] + E&P 감가상각 vs 세무 감가상각 차이: ADS 정액 감가상각(E&P용) > MACRS 가속상각(세무용) → 세무에서 과잉 공제된 감가상각 환원 [ - 차감 항목 ] − 연방소득세 납부액 [IRC §312(f)] − 비현금 이익 (일부 종류) − 세무감가상각 < E&P감가상각인 경우 차이 차감 [ ± 특수 조정 ] ± 즉시상각(Bonus Depreciation / §179) E&P에서 제한적 인정 ± NOL 이월공제 E&P 계산에서 별도 취급 ± 재고자산 방법 차이 (LIFO → FIFO 전환 등) ───────────────────────────────────── E&P_current (당해연도 이익잉여금)
MACRS 가속 감가상각은 세무 과세소득을 줄이지만, E&P 계산에서는 ADS 정액법을 사용해야 하므로 E&P가 과세소득보다 높거나 낮을 수 있음 → 세무 흑자여도 E&P 적자 가능 → ROC 발생

감가상각 괴리: REIT ROC의 수학적 근거

REIT ROC 발생 메커니즘
REIT 순현금흐름(Cash FFO): FFO = Net_Income + Depreciation + Amortization [비현금 비용 환원] E&P 계산 (IRC §857): E&P = Net_Income + ΔDepreciation(E&P - Tax) ≈ Net_Income + (ADS_Depr - MACRS_Depr) 분배금 지급 가능액: Distribution = FFO = Net_Income + Depreciation(MACRS, 대규모) ROC 발생: ROC = Distribution − E&P = (Net_Income + MACRS_Depr) − (Net_Income + ADS_Depr) = MACRS_Depr − ADS_Depr > 0 (MACRS가 더 빠르므로)
결론: REIT의 ROC = 세무 가속 감가상각과 E&P용 정액 감가상각의 차이. 이것은 "감가상각비라는 비현금 세금 방패"를 투자자에게 패스스루(pass-through)하는 메커니즘. → ROC는 이익의 반환이 아니라 세금 방패(tax shield)의 이전

Section 03 — Financial Engineering

금융공학적 관점: ROC의 현금흐름 구조

DCF와 ROC: 투자론적 증명

FINANCIAL ENGINEERING — ROC의 자본 조정 효과
투자자 관점 손익 등식: Total_Return = Income_Return + Capital_Return Income_Return = Dividends_Taxable / Cost_Basis Capital_Return = (Sale_Price − Adj_Cost_Basis) / Original_Cost_Basis ROC 효과: Adj_Cost_Basis(t) = Adj_Cost_Basis(t-1) − ROC(t) → ROC 누적 수령 시: Adj_Cost_Basis(T) = Original_Basis − Σ ROC(t) 매도 시 자본이득: Capital_Gain(T) = Sale_Price − Adj_Cost_Basis(T) = Sale_Price − (Original_Basis − Σ ROC(t)) = [Sale_Price − Original_Basis] + Σ ROC(t) ↑ ROC 수령만큼 자본이득 증가
ROC는 세금을 없애는 것이 아니라 과세 시점을 이연(defer)합니다. 매도 전까지: 비과세 / 매도 후: 자본이득세(15~20%) 적용 → 한국 거주자의 경우 한미조세협약 Art.16으로 미국 자본이득세 면제 가능

REITs/MLPs: 구조적 ROC 발생 메커니즘

⬛감가상각 패스스루

REIT는 부동산 감가상각(MACRS: 27.5년 주거용, 39년 상업용)을 세무상 공제하나 현금유출은 없음. 이 비현금 공제로 인해 E&P < FFO(Funds From Operations). 차이가 ROC로 지급됨.

⬛MLP 자원 고갈

석유·가스 MLP(Master Limited Partnership)는 자원 매장량 감소분(Depletion)을 비현금 비용으로 처리. 운영 현금흐름 > 과세소득 → 초과 분배분이 ROC. IRC §611~613 규정 적용.

⬛폐쇄형 펀드(CEF)

CEF는 관리 수익배분 정책(Managed Distribution Policy)으로 월별 고정 분배를 지급. 시장 하락기 또는 수익 부족 시 원금에서 충당 → ROC. Constructive vs Destructive ROC 구분이 핵심.

ROC 세금 이연 효과 (현재가치 분석)

NPV of Tax Deferral via ROC
즉시 과세 시 세금 납부 현재가치: PV_immediate = ROC × τ_ordinary (배당으로 처리될 경우) ROC로 이연 후 매도 시 세금 현재가치: PV_deferred = (ROC × τ_cg) / (1+r)^T 이연 효과 (Tax Deferral Benefit): Benefit = ROC × [τ_ordinary − τ_cg / (1+r)^T] 수치 예시 (τ_ordinary=37%, τ_cg=20%, r=5%, T=10년): PV_immediate = ROC × 0.37 PV_deferred = ROC × 0.20 / (1.05)^10 = ROC × 0.1228 Benefit = ROC × (0.37 − 0.1228) = ROC × 0.2472 (24.7% 절세 효과)
한국 거주자의 경우: τ_cg = 0% (한미협약 Art.16 자본이득 면제) 적용 가능 →
Benefit = ROC × τ_ordinary = 전액 절세 (단, 한국 국내 양도소득세 별도 검토 필요)

E&P 워크시트 계산 절차

1

과세소득 확인

Form 1120(법인세 신고서) Line 28의 과세소득을 E&P 계산 출발점으로 설정. 이후 IRC §312 조정 항목을 체계적으로 적용.

2

감가상각 재계산 (E&P용 ADS)

세무 신고에서 사용한 MACRS 가속 감가상각을 ADS(Alternative Depreciation System) 정액법으로 재계산. 차이 금액을 E&P에 가산(ADS > MACRS면) 또는 차감(ADS < MACRS면).

3

특수 가산·차감 항목 반영

면세 이자 수입 가산 / 연방소득세 납부액 차감 / 비용불인정 지출 조정 / §179·보너스 상각 조정 등 수십 가지 항목을 IRC §312(a)~(n)에 따라 반영.

4

누적 E&P 계산

E&P_accumulated(T) = Σ E&P_current(t) − Σ Dividends_paid(t) (t=1913 또는 설립년도~현재). LIFO 원칙: 당해연도 E&P 우선 배분(IRC §316(a)).

5

ROC 금액 결정

ROC = Distribution − min(Distribution, E&P_current + E&P_accumulated). 결과값이 양수이면 해당 금액이 ROC이고, Form 1099-DIV Box 3에 표시됨.

개인 투자자 Basis 추적 툴

INDIVIDUAL INVESTOR — Adjusted Basis Tracking
초기 취득 원가 설정: Adjusted_Basis(0) = Purchase_Price + Commission + Fees ROC 수령 시마다 조정: Adjusted_Basis(t) = Adjusted_Basis(t-1) − ROC_received(t) 단, Adjusted_Basis는 0 미만 불가: Adjusted_Basis(t) = max(Adjusted_Basis(t-1) − ROC(t), 0) Excess = max(ROC(t) − Adjusted_Basis(t-1), 0) → 즉시 자본이득 인식 매도 시: Recognized_Gain = Sale_Price − Adjusted_Basis(T) Holding_Period = 결정 → 단기(39.6%) vs 장기(0/15/20%) 세율 적용

Section 05 — QI Reclassification

QI(적격 중개인) 재분류와 ROC 원천징수 정정

한국 거주자가 미국 펀드·REITs에서 ROC를 수령할 때, 한국 증권사(QI 자격 보유 시) 또는 미국 수탁기관이 ROC를 배당(Dividend)으로 오분류하여 30% 또는 15%(한미협약) 원천징수를 부과하는 경우가 발생합니다. 이 경우 QI 재분류 절차를 통해 환급 가능합니다.

QI 시스템의 구조와 ROC 처리

⬛QI란?

외국 금융기관이 IRS와 체결하는 협약(Rev. Proc. 2022-43, 2023 QI Agreement). QI는 원천징수 대리인으로서 Chapter 3(비거주자 원천징수), Chapter 4(FATCA)의 의무를 자체적으로 이행. 원천징수율 풀(withholding rate pool)로 보고 가능.

⬛ROC의 QI 처리 문제

펀드가 연말 전 ROC·배당 비율을 확정하지 못해 일단 전액 배당(Code 06)으로 원천징수 후 연말 Form 8937로 ROC 금액을 공시하는 경우가 많음. 이때 QI가 재분류(reclassification)를 수행하지 않으면 투자자가 초과 납부.

⬛1042-S 소득코드

Form 1042-S Box 1 Income Code: 06=배당, 15=단기자본이득, 16=장기자본이득, 37=기타소득(ROC 처리 시 사용). QI가 배당(06)으로 발급한 1042-S를 정정하여 37로 재발급하면 원천징수 0% 또는 협약세율 적용.

⬛집합 환급(Collective Refund)

QI는 고객을 대리하여 집합 환급(Collective Refund) 신청 가능(2023 QI Agreement §9). 단, PTP(상장 파트너십) 분배 관련 초과 원천징수는 집합 환급 불가 → 투자자 개별 1040-NR 신청 필요.

한국 거주자 특수 고려사항

⬛한미협약 Art.16 자본이득 면제

ROC로 Basis 감소 후 매도 시 발생하는 자본이득: 한미협약 Art.16에 따라 미국에서 원칙적으로 면세 (부동산·PE 관련 제외). 단, 한국 국내에서 양도소득세 부과 가능 여부 별도 검토 필수.

⬛해외금융계좌 신고(FBAR·FATCA)

미국 계좌 잔고 $10,000 초과 시 FBAR(FinCEN 114) 신고 의무. 미국 금융자산 $50,000/$100,000 초과 시 FATCA(Form 8938) 신고. ROC 수령 자체가 추가 신고 의무를 발생시키지는 않으나 계좌 잔고 관리 필요.

⬛한국 종합소득세 처리

미국에서 ROC로 비과세 처리된 금액의 한국 내 세무 처리는 명확한 입법 사례가 부족. 실무상 한국 세무사와 협의하여 처리. ROC 수령액을 취득원가 감소로 처리하는 것이 경제적 실질에 부합.

⬛이중과세 방지 세액공제

ROC 부분에 미국 원천징수가 부과된 경우 → 미국에서 환급 후 한국 소득세 신고 시 외국납부세액공제 불가(환급받았으므로). 환급 전 미국세를 한국에서 공제하면 이중 혜택 문제 발생.
